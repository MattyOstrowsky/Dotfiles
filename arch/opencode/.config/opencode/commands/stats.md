Show token usage and cost statistics using `opencode stats`.

## Usage

Summarize token usage, costs, and model breakdown:

```bash
# Today's stats
opencode stats --days 1

# Last 7 days
opencode stats --days 7

# All time
opencode stats
```

## Output Interpretation

| Metric | What it means |
|--------|---------------|
| input tokens | Tokens sent to the model (your messages + context) |
| output tokens | Tokens generated by the model (responses) |
| total cost | Sum of all API costs in the period |
| model breakdown | Cost per model used |
