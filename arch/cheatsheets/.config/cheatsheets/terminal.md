# Terminal & Fish Shell

## Ghostty
| Shortcut | Action |
|---|---|
| `Ctrl+Shift+H/J/K/L` | New split (left/down/up/right) |
| `Ctrl+Shift+←↑↓→` | Focus split |
| `Ctrl+Shift+E` | Equalize splits |
| `Ctrl+Shift+Q` | Close pane |
| `Alt+Shift+Space` | Quick terminal |

## Fish — FZF
| Shortcut | Action |
|---|---|
| `Ctrl+R` | Search history |
| `Ctrl+Alt+F` | Search directories |
| `Ctrl+Alt+L` | Git log |
| `Ctrl+Alt+S` | Git status |
| `Ctrl+G` | Navi cheatsheet |

## Fish — General
| Shortcut | Action |
|---|---|
| `..` / `...` | cd up 1 / 2 levels |
| `ll` | ls -lah |
| `cat FILE` | bat (syntax highlight) |
| `top` | btop |
| `md FILE` | glow (markdown viewer) |
| `n` | nvim |
