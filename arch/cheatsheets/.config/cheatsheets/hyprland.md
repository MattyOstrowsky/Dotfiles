# Hyprland + Noctalia Keybinds

> **Mod key:** Super (Windows key)

## Launch

| Shortcut | Action |
|---|---|
| `Super + Enter` | Terminal (ghostty) |
| `Super + B` | Firefox |
| `Super + N` | Nautilus (files) |

## Window Management

| Shortcut | Action |
|---|---|
| `Super + Q` | Close window |
| `Super + F` | Fullscreen toggle |
| `Super + V` | Float/tile toggle |
| `Super + P` | Pseudo-tile toggle (dwindle) |

## Navigation (vim-style)

| Shortcut | Action |
|---|---|
| `Super + H` | Focus left |
| `Super + J` | Focus down |
| `Super + K` | Focus up |
| `Super + L` | Focus right |
| `Super + Shift + H` | Move window left |
| `Super + Shift + J` | Move window down |
| `Super + Shift + K` | Move window up |
| `Super + Shift + L` | Move window right |

## Workspaces

| Shortcut | Action |
|---|---|
| `Super + 1..0` | Switch to workspace |
| `Super + Shift + 1..0` | Move window to workspace |
| `Super + mouse_scroll_down` | Next workspace |
| `Super + mouse_scroll_up` | Previous workspace |

## Noctalia Shell

| Shortcut | Action |
|---|---|
| `Super + Space` | Launcher |
| `Super + S` | Control Center |
| `Super + ,` | Settings |

## Media Keys

| Key | Action |
|---|---|
| `Volume Up` | Increase volume |
| `Volume Down` | Decrease volume |
| `Mute` | Toggle mute |
| `Brightness Up` | Increase brightness |
| `Brightness Down` | Decrease brightness |

## Screenshots

| Shortcut | Action |
|---|---|
| `Super + Shift + S` | Area screenshot (select region) |
| `Super + Print` | Full screen screenshot |
| `Shift + Print` | Area screenshot (alternative) |
