# Neovim Keybindings
> `<leader>` = `Space`

## Navigation
| Shortcut | Action |
|---|---|
| `Ctrl+h/j/k/l` | Move between windows |
| `Tab` / `Shift+Tab` | Next / prev buffer |
| `Ctrl+n` | Toggle file tree |
| `<leader>e` | Focus file tree |

## Editing
| Shortcut | Action |
|---|---|
| `Ctrl+s` | Save file |
| `<leader>/` | Toggle comment |
| `<leader>fm` | Format code |
| `jk` (insert) | Escape to normal |
| `;` | Enter command mode |

## Telescope (Fuzzy Finder)
| Shortcut | Action |
|---|---|
| `<leader>ff` | Find files |
| `<leader>fw` | Live grep |
| `<leader>fb` | Find buffers |
| `<leader>fo` | Recent files |
| `<leader>fa` | Find all (incl. hidden) |

## LSP
| Shortcut | Action |
|---|---|
| `gd` | Go to definition |
| `gD` | Go to declaration |
| `<leader>ra` | Rename symbol |
| `<leader>D` | Type definition |
| `<leader>ds` | Diagnostics list |

## Buffers & Windows
| Shortcut | Action |
|---|---|
| `<leader>b` | New buffer |
| `<leader>x` | Close buffer |
| `<leader>h` | Horizontal terminal |
| `<leader>v` | Vertical terminal |
| `Alt+h/v/i` | Toggle terminal |

## Diagnostics / Debug
| Shortcut | Action |
|---|---|
| `<leader>xx` | Toggle trouble |
| `<leader>xw` | Buffer diagnostics |

## Other
| Shortcut | Action |
|---|---|
| `<leader>ch` | Cheatsheet (this menu) |
| `<leader>u` | Undo tree |
| `<leader>wK` | Which-key: all keymaps |
| `Ctrl+c` | Copy whole file |

## File Tree (nvimtree)
| Key | Action |
|---|---|
| `a` | New file/dir |
| `d` | Delete |
| `r` | Rename |
| `y` / `Y` | Copy name/path |
| `R` | Refresh |
| `H` | Toggle dotfiles |
| `Enter` | Open file |
| `Ctrl+v` | Open in vertical split |
