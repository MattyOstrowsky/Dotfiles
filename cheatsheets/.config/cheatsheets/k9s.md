# k9s Keybindings

## Custom Quick-Jump
| Shortcut | Action |
|---|---|
| `Shift+0` | Pods |
| `Shift+1` | Deployments |
| `Shift+2` | Services |
| `Shift+3` | Namespaces |
| `Shift+4` | Helm |
| `Shift+5` | Nodes |
| `Shift+6` | ConfigMaps |
| `Shift+7` | Secrets |
| `Shift+8` | Events |

## Global
| Shortcut | Action |
|---|---|
| `?` / `Ctrl+h` | Help |
| `:` | Command prompt |
| `Ctrl+a` | All namespaces |
| `Ctrl+w` | Wide columns |
| `:ctx` | Switch context |
| `:ns` | Switch namespace |
| `Esc` / `Ctrl+c` | Back / quit |

## Navigation
| Shortcut | Action |
|---|---|
| `j` / `k` | Up / down |
| `g` / `G` | Top / bottom |
| `Ctrl+f` / `Ctrl+b` | Page down / up |
| `Enter` | View resource |
| `/` | Filter |

## Resource Actions
| Shortcut | Action |
|---|---|
| `d` | Describe |
| `e` | Edit YAML |
| `y` | View YAML |
| `l` | Logs |
| `s` | Shell into container |
| `Shift+f` | Port forward |
| `x` / `Ctrl+d` | Delete |
| `Ctrl+k` | Force kill |

## Log View
| Shortcut | Action |
|---|---|
| `t` | Timestamps |
| `w` | Wrap lines |
| `a` | Autoscroll |
| `c` | Clear |
| `/` | Search |
